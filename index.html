Sangi1.0

Sangi is a low cost blind robot which uses minimum possible sensors to perform localization and mapping of the environment

View project on GitHub

Welcome to Sangi1.0 Project Page

Have you ever searched for something in the dark. Have you ever wondered how a blind man moves around in the world. We are building a robot to do just that. Sangi is a Sanskrit word which means a friend, follower and a companion. Our robot Sangi1.0 will navigate around the house blindly and do a variety of tasks.

Our project is build primarily for the purpose of producing a research paper but later we if everything permits, we might go and market our product also.

Overall Structure

Sangi uses a variational approximation with Bayesian Inference to estimate its position in the environment. The likelihood estimation is done using EM algorithm on a Gaussian Scale Mixture. Mapping of the environment is done using an incremental mapping algorithm whose efficiency is increased using a kNN based classification technique.

The lower level algorithms include an EKF to estimate the odometry data from the wheel encoders. The control is achieved using a fuzzy calibrated PID control loop which maintains linear and rotational velocity of the robot. Sonar profiling is done using an ANOVA regression method to give a normal belief of obstacle.

Hardware used in the project are Raspberry Pi to run the higher level algorithms, Arduino UNO for lower level control, a Remote System for fault diagnosis,a Parallax Ping Sensor and motors with encoder attached.
